# 图标文件

此目录应包含应用程序图标：

- `icon.ico` - Windows 图标
- `icon.icns` - macOS 图标
- `32x32.png` - 32x32 PNG 图标
- `128x128.png` - 128x128 PNG 图标
- `128x128@2x.png` - 128x128@2x PNG 图标
- `icon.png` - 通用 PNG 图标

## 临时解决方案

开发阶段可以使用占位图标或使用在线工具生成：
- https://www.icoconverter.com/
- https://iconverticons.com/online/

## 推荐尺寸

- 原始图标应该至少 512x512 像素
- PNG 格式，透明背景
